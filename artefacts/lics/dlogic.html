Digital Logic

Here's an outline of key concepts in digital logic:

Logic Gates

  • Logic Gates
    • w/ Truth Tables
    • eg. AND Gates, OR Gates, NOT Gates, XOR Gates
    • -> Logic Circuits
      • -> Auxillary Circuits
        • eg. Memory Circuits, Arithmetic Circuits, Bitwise Circuits
  • Universal Gates
    • eg. NAND Gates, NOR Gates
    • -> Any Logic Gate

Bit Patterns

  • Bit Patterns
    • -> Bitwise Operations
    • w/ Base-Dependent Representation
      • eg. Binary Representation, Decimal Representation, Hexadecimal Represenation
  • Numbers
    • -> Arithmetic Operations
    • -> Encodable Numbers
      • eg. Two's Complement Encoding, Extended Encoding, Normalised Encoding
